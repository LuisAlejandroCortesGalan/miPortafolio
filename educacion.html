¿Quieres aprender programacion web??

¡Aprende las bases!

aprende lo basico de HTML:

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Su función principal es estructurar el contenido en un sitio web, permitiendo organizar elementos como texto, imágenes, enlaces, formularios y otros componentes visuales en la página.

MDN Web Docs (Mozilla Developer Network)
W3Schools
freeCodeCamp

aprende lo basico de CSS:

CSS (Cascading Style Sheets) es el lenguaje utilizado para dar estilo y diseño a las páginas web creadas con HTML. Permite definir cómo se verán los elementos en la página, incluyendo colores, tamaños, tipografías, espaciado, posicionamiento y otros aspectos visuales.

MDN Web Docs (Mozilla Developer Network) CSS
W3Schools
freeCodeCamp

aprende lo basico de JavaScript:

JavaScript (JS) es un lenguaje de programación que permite agregar interactividad y funcionalidad dinámica a las páginas web. A diferencia de HTML (estructura) y CSS (diseño), JavaScript se ejecuta en el navegador del usuario y permite manipular elementos en tiempo real, responder a eventos (como clics o desplazamientos) y realizar cálculos o modificaciones sin necesidad de recargar la página.

¿Sabes decir qué hace el programa adjunto?

Para hacerlo, seguramente tendrías que poner algún punto de ruptura (breakpoint) y evaluar el valor de las variables donde se toman decisiones.

Código JavaScript

            
            function verificarNumeroPrimo() { 
                let numero = parseInt(prompt("Escoge un número: "), 10);  // Convertir el input a número entero
                if (isNaN(numero)) {
                    alert("Por favor, introduce un número válido.");
                    return;
                }
                
                let esPrimo = false;
                if (numero > 1) {
                    esPrimo = esNumeroPrimo(numero);  // Llamada a la función para verificar si es primo
                }
                alert(esPrimo);
            }
            
            function esNumeroPrimo(numero) {
                let primo = true;
                let divisor = 2;
                
                // Verificamos divisibilidad del número por los valores menores a él
                while (primo && divisor < numero) {
                    if (numero % divisor === 0) {
                        primo = false;
                    }
                    divisor++;
                }
                return primo;
            }

Aplicación de Edificios

La ciudad de Barcelona, como cualquier ciudad, está formada por un número muy grande de edificios. Hay muchas clases de edificios: edificios de viviendas, hospitales, escuelas, bloques de apartamentos, bloques de oficinas, hoteles, cines, etc.

El ejercicio propuesto consistirá en crear una aplicación que contendrá una clase Edificio y varias clases derivadas. No se instanciará ningún objeto de la clase Edificio, sino que se instanciarán objetos de las clases derivadas de esta. Las clases derivadas son: Hotel, Hospital y Cine.

Todos los edificios tendrán ciertas datos comunes (nombre, número de plantas, superficie (m²)) que se darán al crear el edificio y no se modificarán, pero sí que se deberá poder acceder a ellas para leerlas.

Se creará un método limpiar() en la clase adecuada, el cual mostrará por pantalla el tiempo que se ha tardado en limpiar el edificio, teniendo en cuenta que se tarda un minuto por cada 5 m². Por cada minuto de trabajo se pagará un euro diario. Además, como se limpia cada día, para obtener el costo mensual, se deberá multiplicar por 30 (este costo mensual también se mostrará por pantalla).

Además, se implementará un método llamado calcularCostoVigilancia() que mostrará lo que cuesta al mes contratar vigilantes en cada edificio. En cambio, los vigilantes de los cines pueden vigilar 3000 m². El número de asistentes a la sesión no puede ser mayor que el aforo total de la sala.

En los hoteles, cada día pasa el servicio de habitaciones. Se calcula que cada miembro del servicio puede atender 20 habitaciones. Se creará un método que calcule, y muestre por pantalla: a) Cuántas personas son necesarias para atender el servicio de habitaciones en el hotel. b) Cuál es el total necesario para los sueldos de estas personas, teniendo en cuenta que cada persona cobra 1.000 euros al mes.

El “Hotel Pepita”, de 593 m² y 3 plantas. Tiene 12 habitaciones.
